The purpose of a Guide page

Use the Guide content type to provide guidance or instructions on how to do something. This content type allows you to break up large amounts of text into sections with subheadings. This is useful for providing instructions in a clear and accessible format. You should write the guidance or instructions in a logical order so users can find what they are looking for easily. Do not use the Guide content type to create a list of Frequently Asked Questions.

Appearance

A Guide page is made up of sections of text with subheadings. A green in-page navigation box shows on the right-hand side of the page so that users can navigate to the subheading they are looking for. Typecase will create this navigation bar automatically when you add sections to your Guide.
